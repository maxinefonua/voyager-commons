    This page lists the official language(s) for each country code. This is used to map 'name' tags to their equivalent 'name:XX' tags where they are missing and provide better multi-language support. Please do not list all languages used in a country only the official languages that would be used by the 'name' tag. A list of language codes can be found on Wikipedia

    The following list is also used in other projects:

    • opening_hours.js (format: JavaScript object notation, source: countryToLanguageMapping.js).
    • German carto style: https://github.com/giggls/mapnik-german-l10n/blob/master/country_languages.data.in

    If a country uses multiple official languages please make sure to list all of them or it may cause incorrect mappings.
